# homebrew-tap

Personal Homebrew tap of [`no-simpler`](https://github.com/no-simpler).

## Formulae

| Formula | Description                                                                                                     |
| ------- | --------------------------------------------------------------------------------------------------------------- |
| `clc`   | [Claude Cloak](https://github.com/no-simpler/clc) — use Claude Code across Git worktrees without leaving traces |

## Install

```bash
brew install no-simpler/tap/<formula>

Or tap first, then install:

brew tap no-simpler/tap
brew install <formula>
```
